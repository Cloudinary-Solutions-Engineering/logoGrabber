Logo Grabber

Drag the bookmarklet below to your bookmarks bar. Then open any website and click it to find SVG logo candidates, preview them visually, and upload the selected SVG to Cloudinary.

Logo Grabber

Drag this button to your bookmarks bar. If dragging does not work, manually create a bookmark and paste the bookmarklet code from below into the URL field.

First-time setup

  1. Click the bookmarklet on any webpage.
  2. Enter your Cloudinary cloud name.
  3. Enter your unsigned upload preset.
  4. Choose an upload folder, for example grabbed-logos.

What it finds

  • Inline SVG elements.
  • SVGs loaded via image, object, or embed tags.
  • SVG symbol and sprite references where accessible.
  • SVGs inside open shadow roots.

Manual bookmarklet code

javascript:(()=>{let s=document.createElement('script');s.src='https://pglithro-cloudinary.github.io/logoGrabber/script.js?'+Date.now();document.body.append(s)})()

Use an unsigned Cloudinary upload preset only. Do not add API secrets to a bookmarklet or client-side JavaScript file.
